What is CSS3 ?

CSS3 is the latest standard for CSS.

CSS3 is completely backwards-compatible with earlier versions of CSS.

Top 10 New Features in CSS 3

http://www.webreference.com/authoring/css3/index.html

1. Selectors

CSS 3 introduces some new selectors. Using these selectors you can choose DOM elements based on their attributes.

  • [attr=val] - matches a DOM element with the attribute attr and a value with val
  • [attr^=val] - matches a DOM element with the attribute attr and a value starting with val
  • [attr$=val] - matches a DOM element with the attribute attr and a value ending with the suffix val
  • [attr*=val] - matches a DOM element with the attribute attr and a value containing the substring

/* CSS Styles */
div[data-toggle="tooltip"] { color: black; }
p[title^="car"] { color: red; }
img[src*="birth"] { border:3px solid green; }

2. Rounded Corners

CSS 3 introduces border-radius which gives you the same rounded-corner effect.


/* HTML */
<div class="box"></div>

/* CSS Styles */
.box {
  width: 100px;
  height: 100px;
  border: 1px solid #000;
  border-radius: 10px; // shorthand method applies on all corners
}

/* Specific to a certaing corner */
* border-bottom-left-radius || border-bottom-right-radius
* border-top-left-radius || border-top-right-radius

3. Border Image

CSS 3 introduces border-image allows you to specify an image to display instead of a plain solid-colored border.


/* HTML */
<div class="box"></div>

/* CSS Styles */
.box {
  width: 100px;
  height: 100px;
  border: 10px solid transparent;
  border-image: url(http://www.w3schools.com/cssref/border.png) 20% stretch; // round
}

4. Box Shadow

CSS 3 introduces box-shadow allows you to create a drop shadow for an element.


/* HTML */
<div class="box"></div>

/* CSS Styles */
.box {
  width: 100px;
  height: 100px;
  box-shadow: 0px 2px 5px 1px #333; // h-shadow, v-shadow, blur, spread, color
}

5. Text Shadow

CSS 3 introduces text-shadow allows you to add drop shadows to the text on a webpage


/* HTML */
<h1>WEB Development<<h1>

/* CSS Styles */
h1 {
  text-shadow: #333 2px 3px 4px; // h-shadow, v-shadow, blur
}

6. Gradient

Gradient effect is a sleek Web design tool, it can be a drain on resources if not implemented correctly using current CSS techniques.


/* HTML */
<div class="box"></div>

/* CSS Styles */
.box {
  height: 100px;
  width: 200px;
  background: red; /* For browsers that do not support gradients */
  background: -webkit-linear-gradient(#333, #ffb31d); /* For Safari 5.1 to 6.0 */
  background: -o-linear-gradient(#333, #ffb31d); /* For Opera 11.1 to 12.0 */
  background: -moz-linear-gradient(#333, #ffb31d); /* For Firefox 3.6 to 15 */
  background: linear-gradient(#333, #ffb31d); /* Standard syntax (must be last) */
}

/* http://www.w3schools.com/css/css3_gradients.asp */

7. RGBA: Color, Now with Opacity

The RGB property in CSS is used to set colors for different elements. CSS 3 introduces a modification and added opacity manipulation to this property. RGB has been transformed to RGBA (Red Green Blue Alpha channels), which simplifies how you control the opacity of elements.


/* HTML */
<div class="box"></div>

/* CSS Styles */
.box {
  height: 100px;
  width: 200px;
  background-color: rgba(0, 0, 0, 0.5);
}

/* http://www.w3schools.com/css/css3_gradients.asp */

8. Transform (Element Rotation)

CSS 3 introduces transform which enables rotating Web elements on a webpage.


/* HTML */
<div class="box"></div>

/* CSS Styles */
.box {
  height: 100px;
  width: 100px;
  background-color: #333;
  -moz-transform: rotate(30deg);
  -ms-transform: rotate(30deg);
  -o-transform: rotate(30deg);
  -webkit-transform: rotate(30deg);
  transform: rotate(30deg);
}

/* http://www.w3schools.com/css/css3_2dtransforms.asp
 http://www.w3schools.com/css/css3_3dtransforms.asp */
